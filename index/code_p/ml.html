Motivations |
50x50 pix = 2,500 (gray) | 2,500^2/2 ~ 3 mil feature | 2,500*3 = 7,500 (RGB) | 7,500^2/2 ~ 28 mil feature | cortex - кора головного мозга |
BrainPort; Welsh & Blash, 1997 | Haptic Belt - пояс, который передает команды и сообщает направление с помощью тактильных сигналов | Юрий Данилов

Neural Networks | MNIST DATABASE |
neuron takes input(dendrites) = (x1, x2, x3...) -> output(axons) = h(x) = 1/(1+e^th*X) |
theta = "weight" = parameters of the model |
Layers (в скобках): Input -> Hidden -> Output |
a1(2) = g(th10(1)x0 + th11(1)x1 + th12(1)x2 + th13(1)x3) = g(z1(2)) |
a2(2) = g(th20(1)x0 + th21(1)x1 + th22(1)x2 + th23(1)x3) = g(z2(2)) |
a3(2) = g(th30(1)x0 + th31(1)x1 + th32(1)x2 + th33(1)x3) = g(z3(2)) |
h(x) = g(th10(2)*a0(2) + th11(2)*a1(2) + th12(2)*a2(2) + th13(2)*a3(2) |
L1 = 2; L2 = 4 ; th dimensions = 4x3 (2 + bias node(x0)) | a(2) = g(z(2)); z(2) = th(1)*a(1) |

Applications |
h(-4.6) = 0.01 ~ 0 | h(4.6) = 0.99 ~ 1 |
x1 AND x2 | h(x) = g(-30 + 20x1 + 20x2) | h(g(0,0)) = h(-30) = 0 | 0,0; 0,1; 1,0; 1,1 | h(x)=(0,0,0,1) |
(NOT x1) AND (NOT x2) | h(x) = g(10 - 20x1 - 20x2) | h(x)=(1,0,0,0) |
x1 XNOR x2 | (0,0,0,1) OR (1,0,0,0) | h(x)=(1,0,0,1) |
one-vs-all | pedestrian h(x)=[1;0;0;0] | car h(x)=[0;1;0;0] etc |
